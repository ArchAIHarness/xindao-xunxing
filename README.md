# 心道循行

副标题：阳明心学与自然大道双循环要义

《心道循行》是一本关于认知校准、知行合一、循环反馈与自我约束的公开书稿。

它以阳明心学与自然大道为思想参照，讨论人在变化环境中如何观察外部之势、理解自身处境、校准内在动机，并通过行动接受现实反馈。

本书是 ArchAIHarness 的思想外延，服务于“人立法，AI 执行，体系审计”背后的认知说明，但不替代 AI 工程化主线，也不构成宗教、疗愈、命运或人生决策承诺。

## 本书在讲什么

本书尝试讨论一个问题：

> 人在变化的环境中，如何观察外部之势，理解自身之运，觉察当下之遇，校准内在之欲，并通过良知与知行合一形成持续反馈。

它不是对传统经典的权威注解，也不是另立新学，而是作者基于个人阅读、观察与实践形成的一套思想整理。

全书围绕“势、运、遇、欲、良知、知行合一”六个概念展开，把外部环境、个体处境、行动选择与内在判断放在同一个循环框架中讨论。

## 核心脉络

本书围绕两组循环展开：

1. **自然 — 势 — 运**  
   用来观察外部环境、时代变化与个体处境。

2. **遇 — 欲 — 良知 — 知行**  
   用来观察个体如何面对具体情境、校准动机，并通过行动完成反馈。

这两组循环共同指向一个主题：人在现实世界中，不是脱离环境空谈认知，也不是被环境完全决定，而是在观察、判断、行动、反馈中不断校准自己。

## 与 ArchAIHarness 的关系

ArchAIHarness 的主线是 AI 工程化、架构治理与人机协作秩序。

《心道循行》是 ArchAIHarness 的思想外延，不是工程框架，也不替代 `docs`、`framework`、`gateway`、`skills` 等核心公开资产。

它与 ArchAIHarness 的关系主要在于：

- 解释“人立法”背后的认知前提：人需要先判断方向、边界与责任；
- 解释“知行合一”与工程反馈之间的关系：判断必须进入行动，并接受现实校准；
- 解释“循环”思想如何影响长期工程判断：认知、规则、实践、工具、审计需要持续互相反馈。

因此，本仓库服务于 ArchAIHarness 的工程主张，但不改变 ArchAIHarness 以 DDD、AGENTS.md、MCP、Skill Market 和工程审计为核心的主线。
