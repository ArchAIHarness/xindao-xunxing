# AGENTS.md · 心道循行

本仓库用于承载书稿《心道循行：阳明心学与自然大道双循环要义》。

## 协作原则

- 不主动生成正文。
- 不主动扩写目录。
- 不改变作者原稿立意。
- 不将哲学表达改写为宗教宣称、疗愈承诺或权威结论。
- 不引入未经作者确认的历史事实、经典解释或人物评价。
- 不暴露作者私人经历、工作材料、客户信息或未公开内容。
- 如需润色，只做语言清晰化、错别字修正和结构提示。
- 涉及阳明心学、道家思想、传统文化概念时，必须标记不确定处，并建议作者复核。

## 公开边界

- 本仓库为公开仓库，写入内容默认可被公开访问。
- 作者提供的底稿如包含私人经历、工作材料、客户信息、内部系统、未公开指标或敏感细节，必须先脱敏再写入。
- 不把个人修身观点包装为医学、心理治疗、投资、职业决策或命运承诺。

## 书稿处理规则

- 作者明确要求“归档”“写入 manuscript”“整理成章节文件”时，才可将底稿写入仓库。
- 未经作者确认，不新增章节、不补写缺失段落、不替作者推演经典义理。
- 可以在作者要求下做错别字检查、标点统一、章节拆分、目录索引、版本整理。
